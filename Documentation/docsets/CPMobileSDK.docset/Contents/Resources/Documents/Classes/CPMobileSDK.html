CPMobileSDK

public class CPMobileSDK : NSObject

The main mobile SDK class. Use this class’ methods to start the mobile SDK, pass URLs to the SDK and obtain the application encryption key.

Starting the SDK
  • startSDKWithOptions(_:)

    Call this method to initialize the SDK. Usually, you should call this method in UIApplicationDelegate.application(_:didFinishLaunchingWithOptions:).

    Seealso

    CPMobileSDKApplicationMain(_:_:_:_:_:)

    Declaration

    Swift

    public class func startSDKWithOptions(options: CPMobileSDKOptions?)

    Parameters

    options

    An options object, defining the behavior of the mobile SDK.

URL Handling
  • handleSDKOpenURL(_:options:)

    Call this in UIApplicationDelegate.application(_:openURL:options:) method to handle SDK-related URLs.

    Declaration

    Swift

    public class func handleSDKOpenURL(url: NSURL, options: [String : AnyObject]) -> Bool

    Parameters

    url

    The URL resource to handle.
    options

    A dictionary of launch options.

    Return Value

    Returns true if handled by SDK, false if URL is unrelated to SDK and should be handled by the application.

  • handleSDKOpenURL(_:sourceApplication:annotation:)

    Call this in UIApplicationDelegate.application(_:sourceApplication:annotation:) method to handle SDK-related URLs.

    Deprecated

    Use CPMobileSDK.handleSDKOpenURL(_:options:)

    Declaration

    Swift

    public class func handleSDKOpenURL(url: NSURL, sourceApplication: String, annotation: AnyObject) -> Bool

    Parameters

    url

    The URL resource to handle.
    sourceApplication

    The bundle ID of the app that is requesting your app to open the URL (url).
    annotation

    A property list object supplied by the source app to communicate information to the receiving app.

    Return Value

    Returns true if handled by SDK, false if URL is unrelated to SDK and should be handled by the application.

Encryption Key
  • encryptionKey()

    The encryption key for the application. To listen when this key becomes available and unavailable, register to CPMobileSDKEncryptionKeyAvailableNotification and CPMobileSDKEncryptionKeyUnavailableNotification notifications.

    Declaration

    Swift

    public class func encryptionKey() -> NSData?

    Return Value

    The application encryption key, if available, otherwise returns nil.
