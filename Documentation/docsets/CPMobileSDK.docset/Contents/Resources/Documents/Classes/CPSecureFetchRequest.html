CPSecureFetchRequest

public class CPSecureFetchRequest : NSFetchRequest

A subclass of NSFetchRequest for performing searches with predicates referencing secure properties.

Secure fetch requests contain two predicates; predicate for constraining non-secure properties, and securePredicate, for constraining secure properties.

Swift

let secureFetchRequest = CPSecureFetchRequest(entityName: "Entity")
secureFetchRequest.predicate = NSPredicate(format: "integerProperty >= 5")
secureFetchRequest.securePredicate = NSPredicate(format: "secureStringProperty CONTAINS '7'")
  • securePredicate

    The predicate is used to constrain the selection of objects the receiver is to fetch. For more about predicates, see Predicate Programming Guide.

    Use this predicate to constrain secure properties (properties transformed using CPSecureValueTransformer and subclasses).

    Note

    While you can use this property to constrain all properties of your objects, it is not recommended. For performance reasons, make sure to constraints all non-secure properties using predicate, and only constrain secure properties using securePredicate.
