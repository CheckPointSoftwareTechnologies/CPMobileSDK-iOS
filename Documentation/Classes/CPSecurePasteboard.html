CPSecurePasteboard

public class CPSecurePasteboard : UIPasteboard

The CPSecurePasteboard class is a secure, encrypted UIPasteboard subclass. Once encryption key is set, objects stored using instances of CPSecurePasteboard are encrypted before storage. An external message is provided for the users if they paste content outside of secured applications.

  • setEncryptionKey(_:)

    Sets the encryption key to use with CPSecurePasteboard instances. Setting the encryption key to nil will cause instances of CPSecurePasteboard to store objects in clear.

    Declaration

    Swift

    public class func setEncryptionKey(encryptionKey: NSData?)

    Parameters

    encryptionKey

    The encryption key to use with CPSecurePasteboard instances.
  • setExternalMessage(_:)

    The external message to display to users if they paste content outside of secured applications.

    By default, the external message is Pasteboard secured by Check Point.
