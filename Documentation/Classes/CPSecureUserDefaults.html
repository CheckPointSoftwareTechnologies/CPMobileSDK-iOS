CPSecureUserDefaults

public class CPSecureUserDefaults : NSUserDefaults

The CPSecureUserDefaults class is a secure, encrypted NSUserDefaults subclass. Once encryption key is set, objects stored using instances of CPSecureUserDefaults are encrypted before storage.

  • setEncryptionKey(_:)

    Sets the encryption key to use with @c CPSecureUserDefaults instances. Setting the encryption key to nil will cause instances of CPSecureUserDefaults to store objects in clear.
