The following global variables are available globally.

    public let CPMobileSDKDidLockApplicationNotification: String
  • CPMobileSDKDidUnlockApplicationNotification

    Declaration

  • CPMobileSDKDisableSecurityNotification

    Posted when user taps the Try Without Capsule Workspace button. Only applicable if CPMobileSDKOptions.allowAppUsageWithoutCapsuleIntegration is enabled.

    Declaration

    Swift

    public let CPMobileSDKDisableSecurityNotification: String
  • CPMobileSDKEncryptionKeyAvailableNotification

    Posted when the encryption key becomes available, once the user has authentication with Capsule Workspace.

    Declaration

    Swift

    public let CPMobileSDKEncryptionKeyAvailableNotification: String
  • CPMobileSDKEncryptionKeyUnavailableNotification

    Posted when the encryption key becomes unavailable, usually if user has logged out in Capsule Workspace, or some other issue has come up and requires the user’s reauthentication.

    Declaration

    Swift

    public let CPMobileSDKEncryptionKeyUnavailableNotification: String

    A key whose value is a NSData object representing the encryption key for the application. This key is used with CPMobileSDKEncryptionKeyAvailableNotification notifications.
